WeensyOS

WeensyOS was essentially creating a very basic and small operating system. This project was centered around implementing the basic virtual and physical memory architecture as well as the system calls fork and exit.

The first step of the project was making sure that no process could write to the kernel or allocate pages of memory from the kernel.

The second step of the project was making sure that each process could not write to the other processes running on the operating system.
